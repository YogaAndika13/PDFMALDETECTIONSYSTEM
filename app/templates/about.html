About This Project

This web application is designed to detect malware in PDF files using a machine learning approach.

Methodology

The core of this system is a Stacking Ensemble model combining a Multi-Layer Perceptron (MLP) and a Support Vector Machine (SVM). This model has been pre-trained on a comprehensive dataset of benign and malicious PDF files.

The detection process involves:

  1. Feature Extraction: Static and dynamic features are extracted from the uploaded PDF files. These features capture various characteristics of the PDF structure and content that are indicative of malicious intent.
  2. Prediction: The extracted features are fed into the pre-trained MLP-SVM stacking model, which classifies the file as either 'Benign' (safe) or 'Malicious'.
  3. Explainability: To provide insight into the model's decision, we use SHAP (SHapley Additive exPlanations). This technique highlights which features were most influential in the prediction for a given file.

Limitations

  • The accuracy of the detection is dependent on the pre-trained model and the features it was trained on.
  • The system may not be able to detect all types of PDF-based malware, especially novel or highly obfuscated threats.
  • The feature extraction process might not handle all PDF formats or encrypted files perfectly.
